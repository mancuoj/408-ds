## Features

- 🍓 09~22年真题
- 🍒 暴力解 + 最优解
- 🥭 仿照王道书上的写法，含注释
- 🍉 [GoogleTest](https://github.com/google/googletest) 全面测试
- 🍇 真题题目 + 评分标准

## Usage

推荐使用 CLion 打开本项目，可在 `test_ds.cpp` 文件中添加更多测试：

```c++
TEST(xx, xx) {
    
    ...
    
    EXPECT_EQ(add(1, 2), 3);
    EXPECT_TRUE(is_exist());
}
```
